# 莫高窟橘色筆記 PWA

已更新至補入第 130 窟版本。

## 部署到 GitHub Pages

```bash
git clone https://github.com/mikemm5216/mogao-pwa.git
cd mogao-pwa
cp -R /path/to/mogao-pwa-final-130/* .
git add .
git commit -m "Update Mogao PWA with Cave 130"
git push origin main
```

之後到 GitHub Pages 使用 HTTPS 開啟：

https://mikemm5216.github.io/mogao-pwa/

iPhone Safari：分享 → 加入主畫面。第一次開啟後即可離線使用。
